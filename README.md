## How Auto-Update Works

- `scripts/generate_readme.py` fetches your repositories from the GitHub API.
- It rebuilds the dashboard sections below in order, with categories and newest repos.
- `.github/workflows/update-readme.yml` runs this automatically every day and on pushes to `main`.
